# Accessibility Checklist

## Visual
- [ ] Sufficient color contrast
- [ ] Legible font sizes
- [ ] Avoid text embedded in images

## Navigation
- [ ] Logical heading structure
- [ ] Everything reachable via keyboard
- [ ] ARIA labels for interactive controls

## Media
- [ ] Alt text for images
- [ ] Captions for video (if any)

## Data App
- [ ] Table headers labeled
- [ ] Filters have labels
- [ ] Charts include text-based summaries
