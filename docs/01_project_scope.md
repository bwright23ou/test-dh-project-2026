# Project Scope & Phasing

## Phase 1: Discovery & Planning
- Complete discovery checklist
- Content inventory
- Define audiences
- Define success criteria

## Phase 2: Data Intake & Cleaning
- Add raw dataset(s) to `data/raw/`
- Run initial exploration notebook
- Identify inconsistencies and cleanup steps
- Produce data dictionary (03_data_dictionary.md)

## Phase 3: Content Extraction & Site Build
- Export or scrape WordPress (or other CMS)
- Convert content to 11ty markdown + frontmatter
- Create site structure document (06_site_structure.md)

## Phase 4: App Requirements & Prototypes
- Write user stories (04_user_stories.md)
- Define app functionality (05_app_requirements.md)
- Build Streamlit prototype

## Phase 5: Integration & Deployment
- Deploy 11ty static site
- Deploy Streamlit app
- Embed or reverse proxy app into site

## Phase 6: Licensing & Citation
- Document chosen licenses (07_licensing_plan.md)
- Prepare Zenodo registration (08_citation_plan.md)

## Phase 7: QA & Handoff
- Accessibility review (09_accessibility_checklist.md)
- Sustainability & handoff doc (10_sustainability_handoff.md)
- Final walkthrough with project owner
